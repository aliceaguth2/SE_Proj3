# EcoBites

## Our Mission

At EcoBites, we are redefining food delivery with sustainability at its core, empowering customers to make every meal an eco-conscious choice. Through our food delivery platform, we provide flexible eco-friendly options, from reusable or compostable packaging to rewards for low-carbon delivery methods, while encouraging shared orders to reduce trips and enabling a marketplace for cancelled orders to minimize food waste. Our goal is to create a food delivery experience that benefits people, communities, and the planet—one conscious meal at a time.

## Key Features

### 🌱 For Customers
- **Eco-Friendly Ordering**: Choose sustainable packaging options (reusable, compostable, minimal) and earn eco-rewards
- **Seasonal Highlights**: Support local, seasonal ingredients with bonus reward points
- **Order Combining**: Reduce delivery trips by combining orders with neighbors (+20 eco-points)
- **Cancelled Order Marketplace**: Bid on cancelled orders to reduce food waste
- **Restaurant Reviews**: Rate restaurants and help others make informed choices
- **Impact Tracking**: Monitor your eco-rewards and environmental impact
- **Profile Preferences**: Save your packaging preferences and delivery addresses

### 🍽️ For Restaurants
- **Menu Management**: Create and manage menu items with seasonal highlights
- **Eco-Packaging Options**: Showcase your sustainable packaging choices
- **Rating System**: Build reputation through customer reviews and ratings
- **Review Responses**: Engage with customer feedback directly
- **Order Management**: Streamlined order workflow from placement to preparation

### 🚗 For Delivery Drivers
- **Vehicle Incentives**: Earn extra points for eco-friendly vehicles (electric: +15pts, hybrid: +10pts, gas: +5pts per delivery)
- **Order Dashboard**: View available, current, and completed orders
- **Flexible Acceptance**: Choose which orders to accept
- **Performance Tracking**: Monitor deliveries and earnings
- **Location Sharing**: Share location with customers during delivery

## How It Works

### Customer Journey
1. **Browse** eco-friendly restaurants with ratings and reviews
2. **Select** menu items, including seasonal highlights for bonus points
3. **Choose** sustainable packaging options to earn eco-rewards
4. **Combine** orders with neighbors to reduce delivery trips
5. **Track** your order status in real-time
6. **Review** restaurants after delivery to help the community

### Restaurant Workflow
1. **Create** menu items with seasonal highlights and eco-packaging options
2. **Receive** orders from customers
3. **Manage** order status from acceptance to preparation
4. **Respond** to customer reviews and build your reputation
5. **Monitor** ratings and customer feedback

### Driver Workflow
1. **View** available orders ready for pickup
2. **Accept** orders and navigate to restaurant
3. **Pick up** prepared orders
4. **Deliver** to customers with location tracking
5. **Earn** eco-rewards based on your vehicle type

### Marketplace for Sustainability
1. **Cancelled Orders**: When customers cancel, orders enter the marketplace
2. **Bidding System**: Other customers can bid to claim cancelled orders
3. **Waste Reduction**: Original customer accepts best bid, preventing food waste
4. **Order Reassignment**: Winning bidder gets the order at their bid price

## Impact

- **Customers:** Make environmentally conscious choices while enjoying convenient food delivery. Track personal impact through eco-rewards, participate in sustainable practices, and contribute to reducing food waste through the cancelled order marketplace.

- **Restaurants:** Partner with us to showcase locally-sourced, seasonal menu options and eco-friendly practices. Build reputation through ratings and reviews while attracting conscious customers who value sustainability.

- **Delivery Drivers:** Contribute to reducing carbon footprint through incentivized low-emission delivery methods. Earn rewards for using electric vehicles, bicycles, or optimized routes while supporting environmental responsibility.

## Technology Stack

- **Frontend**: React 18, Vite 7, Tailwind CSS 4, React Router DOM
- **Backend**: Node.js 18+, Express.js 5, MongoDB with Mongoose 8
- **Authentication**: JWT with bcrypt password hashing
- **Testing**: Jest + Supertest (backend), Vitest + React Testing Library (frontend)
- **Code Quality**: ESLint, Prettier, Babel
- **CI/CD**: GitHub Actions with automated testing and code coverage
- **Optional**: Geocoding API for address coordinate lookup

## Quick Start

### Prerequisites
- Node.js 18+
- MongoDB (local or Atlas)
- npm 8+
- Git

### Installation

```bash
# Clone repository
git clone <repository-url>
cd proj2

# Install dependencies
cd Ecobites/server && npm install
cd ../client && npm install

# Set up environment variables (see INSTALL.md)
cd ../server
cp .env.example .env
# Edit .env with your configuration

# Seed database
npm run seed

# Start backend (Terminal 1)
npm start

# Start frontend (Terminal 2)
cd ../client
npm run dev
```

Visit http://localhost:5173 to access the application.

For detailed setup instructions, see [INSTALL.md](INSTALL.md).

## Eco-Rewards System

### Earning Points
- **Packaging Choices**:
  - Reusable: +10 points
  - Compostable: +5 points
  - Minimal/Standard: 0 points
- **Seasonal Items**: Variable points (set by restaurant)
- **Order Combining**: +20 points per participant
- **Driver Incentives**:
  - Electric vehicles: +15 points per delivery
  - Hybrid vehicles: +10 points per delivery
  - Gas vehicles: +5 points per delivery

### Using Rewards
- Track your environmental impact
- View reward history
- Future: Redeem for discounts and special offers
